== Description ==
Amazon Auto Linker finds keywords on your site and hyperlinks them with affiliate links. As a result you will get commission from the sales referred from your links.

== Installation ==
1. Upload the plugin through `Plugins > Add New > Upload` interface or upload to wp-content/plugins directory using FTP.
2. Activate the plugin once uploaded from the wordpress plugin administration page.
3. Go to Amazon Auto Linker settings panel and enter your Amazon affiliate ID.
